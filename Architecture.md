Type: CLI Analytics Tool. Pattern: Pipe-and-Filter. Components: 1. Data Source (Local Git CLI via Subprocess). 2. Aggregator (In-memory grouping using collections.defaultdict). 3. Formatter (String-join based reporting). Error Handling: Returns sentinel strings on subprocess failure to prevent stack trace leaks to end-user. Optimization: Uses generator expressions for O(n) line splitting and list unpacking for report building.
